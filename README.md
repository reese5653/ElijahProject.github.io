# ElijahProject

A simple navigation system with 16 buttons linking to different pages.

## Features

- **16 Interactive Buttons**: Each button links to a unique page
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Beautiful gradient styling with smooth hover effects
- **Easy Navigation**: Back button on each page to return to the main menu

## Project Structure

```
ElijahProject/
├── index.html          # Main page with 16 navigation buttons
├── page1.html          # Individual pages (1-16)
├── page2.html
├── ...
├── page16.html
└── styles.css          # Shared styling for all pages
```

## How to Use

1. Open `index.html` in a web browser
2. Click any of the 16 buttons to navigate to that page
3. Click the "Back to Home" button to return to the main page

## Local Development

To run locally with a simple HTTP server:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080` in your browser.
