# Elijah Project School of Ministry

A comprehensive ministry training website built with TypeScript, Next.js, and Tailwind CSS. This platform provides 16 different training modules designed to equip believers for effective ministry and service.

## Features

- **16 Ministry Training Modules** covering essential topics from Biblical Foundations to Media & Technology
- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Modern TypeScript** - Fully typed with TypeScript for reliability and maintainability
- **Next.js Framework** - Built with the latest Next.js App Router for optimal performance
- **Tailwind CSS** - Beautiful, customizable styling with dark mode support

## Getting Started

### Prerequisites

- Node.js 18.x or later
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/reese5653/ElijahProject.git
cd ElijahProject
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to see the application.

## Available Scripts

- `npm run dev` - Starts the development server on http://localhost:3000
- `npm run build` - Builds the application for production
- `npm start` - Starts the production server
- `npm run lint` - Runs the linter to check for code quality issues
